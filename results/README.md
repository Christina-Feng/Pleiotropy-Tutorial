# Results Folder

This folder stores the intermediate data structures and analytical results generated by executing the tutorial workflow steps. Files are saved as R serialized objects (`.rds`) and as a compiled HTML report.

## Output Files

* **`prepared_summary_statistics.rds`** (Step 01): Aligned matrices and annotations ready for use in all pleiotropy packages.
* **`asset_results.rds`** (Step 02): Results from subset-based meta-analysis using `ASSET`.
* **`placo_results.rds`** (Step 03): Outputs from composite null hypothesis testing using `PLACO`.
* **`gpa_results.rds`** (Step 04): Output from Genetic Pleiotropy Analysis with functional annotations using `GPA`.
* **`cpbayes_results.rds`** (Step 05): Analysis outputs from the Bayesian cross-phenotype model using `CPBayes`.
* **`gcpbayes_results.rds`** (Step 06): Gene-level Bayesian cross-phenotype results using `GCPBayes` evaluated on the *BCAT1* gene.
* **`pleiotropy_workflow.html`** (Step 07): A complete HTML report compiling all tutorial vignettes results.
